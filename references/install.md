# Install

Clone or copy the `arxiv-packager` skill folder into your Codex skills directory:

```bash
cp -R arxiv-packager ~/.codex/skills/
```

Restart Codex after installing so the new skill is discovered.

For workspace packaging, invoke the skill with the main TeX source file and output stem:

```text
Use $arxiv-packager to package `main.tex` with output stem `paper`
```
